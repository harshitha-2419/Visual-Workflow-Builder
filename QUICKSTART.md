# Quick Start Guide

## Get Started in 3 Minutes! 🚀

### Step 1: Install & Run (30 seconds)

```bash
cd workflow-builder
npm install
npm run dev
```

Open http://localhost:5173

### Step 2: Build Your First Workflow (2 minutes)

1. **Drag** "Manual Trigger" from sidebar → canvas
2. **Drag** "HTTP Request" below it
3. **Connect** them (click bottom of trigger → top of HTTP Request)
4. **Click** HTTP Request node → configure in right panel
5. **Click** "Validate" button → check for errors
6. **Click** "Execute" button → watch it run!

### Step 3: Explore Features (30 seconds)

- Press `Ctrl+Z` to undo
- Click "Export" to save
- Click moon icon for dark mode
- Drag more nodes to build complex workflows

## That's It! 🎉

You now have a working workflow automation builder!
